# 物品空间

## 命名空间

你肯定会注意到,ItemsAdder使用的是**命名空间**来识别它所管理的大多数物品.
一个**命名空间**是一个**组合**.在这里是一组物品/方块/怪物.  
通过命名空间,你可以非常容易地找到一个特定的物品、声音、方块......来自哪里.

### Example

所有的**realcraft**物品都在**realcraft**命名空间下,所以当你使用`/iaget`指令时,你可以看到所有物品ID都以`realcraft：`为开头。
