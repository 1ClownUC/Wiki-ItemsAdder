### 末地特性

建议**不要在**末地**生成**自定义树,因为有一些**特性**
非常感谢

### 原版树的特性

有时树木会产生虚拟的叶子,这是一个已知的特性,我没有解决这个特性的办法.这似乎是Minecraft/Spigot的一个bug,在IA使用时有时并没有被触发.

### 大树会导致卡顿

大树可能会导致生成时会产生一些卡顿,建议只使用小树的类型.
