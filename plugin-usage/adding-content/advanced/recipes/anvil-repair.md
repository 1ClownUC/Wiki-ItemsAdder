# Починка на наковальне

## Example

```yaml
anvil_repair:
    emerald_sword:
        permission: itemsadder.emerald_sword
        ingredient: EMERALD
        item: itemsadder:emerald_sword
```

In this example I created an`anvil_repair` recipe called `emerald_sword`
