## Mipmap 问题 \(和材质大小相关\)

{% hint style="warning" %}
如果你的材质图片大小不是偶数，Minecraft 会禁用 mipmap 功能。

### 文件名和物品名

{% hint style="danger" %}
物品名（即配置文件中的**names**），命名空间（即配置文件中的**namespaces**），材质文件名（即**png**类型的文件），模型文件名（即**json**类型的文件）
以上均**不**要使用 **大写字母** 或者是 **特殊字符**
{% endhint %}
